# 🔐 Python Password Manager

A command-line password manager built with Python for storing, viewing, searching, and deleting website credentials.

This project stores credentials in a text file and was created to practice Python file handling, program organization, and user interaction.

> **Note:** This is a learning project. Passwords are stored in plain text and are **not encrypted**.

---

## Features

- ➕ Add website credentials
- 👀 View saved passwords
- 🔍 Search passwords by website
- ❌ Delete saved passwords
- 💾 File-based storage
- ✅ Input validation
- ⚠️ Error handling for missing files

---

## Technologies Used

- Python 3

---

## How to Run

Clone the repository:

```bash
git clone https://github.com/TheAbdullahSaleem/python-password-manager.git
```

Run the program:

```bash
python main.py
```
